# Electric Vehicle Population Analytics Dashboard (Tableau)
## Overview

The Electric Vehicle Population Analytics Dashboard is an interactive Tableau project designed to analyze the growth and distribution of electric vehicles across different states, manufacturers, models, and eligibility programs.

The dashboard enables users to monitor EV adoption trends, compare Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs), explore geographic distribution, and identify the leading vehicle manufacturers and models. Interactive filters allow users to analyze the data by model year, EV type, state, and Top N rankings.

## Key Insights
EV adoption has grown significantly over the years, reflecting increasing market acceptance.
Battery Electric Vehicles (BEVs) represent the majority of registered electric vehicles.
A small number of manufacturers account for a large share of total EV registrations.
Vehicle adoption is concentrated in specific states, highlighting regional differences.
CAFV eligibility analysis provides insight into incentive program coverage across the EV population.
Interactive filtering allows users to compare trends across years, vehicle types, and geographic regions.
